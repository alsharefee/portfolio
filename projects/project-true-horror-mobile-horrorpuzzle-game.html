True Horror: Mobile Horror/Puzzle Game

Release Date: 2018

Mobile Horror Puzzle

The trailer for the mobile IOS and Android game, a puzzle and horror experience "True Horror".
In a house that has been touched by a demonic presences, lives a family that can't leave their home and in there happens horrific things that can't be imagined.

Horror game for mobile that uses the highest graphics possible on mobile. The game also uses speech recognition to solve puzzles, Face camera to add the player face inside the game and AR while game play.
