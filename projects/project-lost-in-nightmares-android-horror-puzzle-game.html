Lost In Nightmares: Android Horror / Puzzle Game

Release Date: 2013

Unity (JavaScript) Android Horror Puzzle

My first ever released game on Google Play on Android system in 2013. Made using Unity3D and JavaScript which was Unity default scripting language at that time.
The game is a horror / puzzle game where players can move in FPS. It reached 70,000+ installations in the first 3 months. 10 years later, Google play required that I update the game to support new android versions, which I didn't do and it got removed from play store.

The game can only be downloaded by third party websites like this: https://lost-in-nightmare.en.aptoide.com/app (Though I don't recommend downloading it from third party websites as they could have replaced it with a malware or something bad for your device.)
