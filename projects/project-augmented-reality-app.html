Augmented Reality Space Door/Portal (AR)

Release Date: 2017

Unity (C#) Apple ARKit API Integration

This is an Augmented Reality app inside a mall, using Unity3D and Apple ARKit I made while working in GamfiedLabs. In the video you can see it was to motivate malls to use AR in their marketing. I believe at that time Apple was still developing ARKit, and so I had to install a beta iOS operating system to be able to develop for it.
